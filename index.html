Los pasos a seguir y el código, como ejemplo, serían los siguientes:

  1. Definir el estilo de las páginas:
    • Anabel crea un documento HTML con un título principal y un botón.
    • Aplica estilos CSS a los elementos que garanticen la consistencia visual.

  2. Estructura del documento HTML:
    • Incluye los metadatos en la sección <head> del documento.
    • Define estilos CSS dentro de la etiqueta <style> para el cuerpo del documento, el título principal y el botón.
    • Incluye los elementos visibles en la sección <body> del document.

  3. Código HTML

La idea de Anabel es crear una página web con un título, un estilo de guía y un botón. El código comienza con la declaración del tipo de documento y el idioma. Luego, en la sección <head>, se establecen metadatos como el juego de caracteres y la escala inicial. También se define un estilo en línea que establece la fuente del cuerpo, el color primario y los estilos para el botón. En el <body>, se utiliza el título principal con la clase "color-primary" para aplicar el color azul definido en el estilo. También se crea un botón con la clase "button" que hereda los estilos definidos en el estilo en línea y cambia su color de fondo al pasar el cursor sobre él.
